# Analytics Engineering

This folder contains codes and homework of Data Engineering Zoomcamp Week 4.

Folder description
- `homework`: folder for homework
- `taxi_rides_ny`: dbt project folder containing ingestion script, SQL models, YAML configuration files, and tests that organize data transformations.
- `pyproject.toml`: configuration file for uv packaging library
- `uv.lock`: records if exact version of dependencies in this folder
