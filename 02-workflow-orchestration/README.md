# Workflow Ochestration

This folder contains codes and homework of Data Engineering Zoomcamp Week 2.

We use Kestra for workflow orchestration in this week.

Folder description
- `homework`: Folder containing homework solutions and codes
- `workflows`: Folder containing Kestra's workflow (`.yaml` file) used in this week
- `docker-compose.yaml`: Docker compose file to spin up Kestra and PostgreSQL
- `setup_secret.sh`: Set up secret as an envorinmental variable to parse into Kestra
